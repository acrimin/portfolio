Corgi Rambo


Technologies: C++, SDL

Description: A 2D shooter game. The player is a corgi fighting tanks to survive. This program is written in C++ and uses Simple DirectMedia Layer (SDL). The program uses data driven programming contained in game.xml. Most settings can be changed in this file.

Controls:
w: jump
a d: left, right
shift: sprint
s: crouch
space: shoot
e: switch gun
F1: toggle hud
r: restart
q: quit

Github: Corgi Rambo

Download: Corgi Rambo

Instructions:

unzip asg6.zip
cd asg6
make
./run

Step By Step


Technologies: Android Studio, Java, MySQL, PHP, Javascript, HTML

Description: Clemson Life is a program for students with intellectual disabilities. The app Step By Step was designed to assist in their day to day lives. It was found that these students could be very close to being independent but would sometimes be confronted with problems they wouldn't understand how to solve on their own. Step By Step serves to remedy this problem. This application connects to an external database where the faculty can add specific tasks for students. Students access the app and based on their location, the app displays relevent tasks. After selecting a task the student will access step by step instructions.

Universal Media Player


Technologies: Java, Java Swing

Description:

Download: Media Player

Instructions: A media player that gets information from a universal media server. This media player has three different users with different levels of permissions. The idea being to censor music from different audiences. The admin (password: 9999) has control over the accounts of user1 (password: 1111) and user2 (password: 2222). This media player is written in Java using Swing for the GUI.

unzip mediaPlayer.zip
cd mediaPlayer

To Start Universal Media Server

cd ums
java -Dums.profile.path=. -cp ums.jar net.pms.PMS

To Start Media Player

chmod +x gradlew
./gradlew build
./gradlew run

Tangible Poster Project


Technologies: Python, Kivy, OSC

Description: This program was designed to assist in the demonstration of a new style of 3D brain modeling. The goal was to create a Tangible Interface to interact with this information rather than the traditional poster. The program used two Microsoft Surface Studios to allow full touch navigation. The program allowed placement of capacitive touch knobs onto the Surface to interact with the model. This project was completed using Python with Kivy for the Graphical User Interface and Open Sound Control (OSC) for the networked communication.

Github: Tangible Poster Project

Download: Tangible Poster Project

Instructions:

unzip tangible.zip
cd P3
python title.py #for both vertical and horizontal

3D Ray Tracer


Technologies: C++
